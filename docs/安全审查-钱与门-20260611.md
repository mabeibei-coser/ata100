## 修复清单(与 ASG100 同款移植 + 1 项特有)

1. **P1-1** trust proxy true→1:防伪造 X-Forwarded-For 绕过 IP 限频(nginx 实测为追加模式,漏洞坐实)
2. **P1-2** 验证码 Math.random → crypto.randomInt(CSPRNG)
3. **P2-1** notify 加实付金额一致性校验(不符拒入账返 500)
4. **P2-2** verifyNotify 加 ±5 分钟时间戳窗口(防重放)
5. **P2-3** 公众号 env 齐全时拒绝 FAKE_CODE(防手动塞假 openid)
6. **ATA 特有** GET /api/legal/:type 加 hasOwnProperty 守门(防 __proto__/constructor 命中原型链;原实现无实际危害,规范化)
7. 文档:session.js cookie path 注释修正(path=/ 是 A500/A800 共享的必然要求)
